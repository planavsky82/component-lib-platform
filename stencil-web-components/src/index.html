Get Started with Your Design System

This is the start point for your design system development platform.

Before you start working ...

  1. Create your project and author name if you haven't done so already but editing your root-level package.json.
    2. {
     "name": "my-design-system",
     ...
     "author": "",
     ...
    }
  2. Also change the name property in the /stencil-web-components/package.json and /stencil-react-wrappers/package.json to something that fits your design system convention.
  3. Finally, make sure the componentCorePackage property in the /stencil-web-components/stencil.config.ts matches the name property in /stencil-web-components/package.json.

You can start learning how to develop the design system by editing this page (/stencil-web-components/src/index.html). The following assets will help:
